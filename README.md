    购物商城全栈项目

     项目介绍
基于 Vue3 + Element Plus + Node.js + Express + MySQL 开发的全栈购物商城，包含完整的前后端功能，适合作为求职作品集展示。

   🛠️ 技术栈
    前端
  框架：Vue 3 + Vite
  UI 组件库：Element Plus
  状态管理：Pinia
  路由：Vue Router
  网络请求：Axios

    后端
  框架：Node.js + Express
  数据库：MySQL
  接口规范：RESTful API
  身份验证：JWT

      核心功能
1.用户模块：登录注册、个人信息管理

2.商品模块：商品列表、商品详情、分类筛选

3.购物车：添加商品、修改数量、删除商品

4.订单模块：订单创建、订单查询、订单管理

5.响应式设计：适配 PC 端多尺寸屏幕
